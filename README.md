# unity-the-bridge-of-gaya

Unity 프로젝트: The Bridge of Gaya

## 설명
이 프로젝트는 Unity 엔진을 기반으로 한 "The Bridge of Gaya" 게임/애플리케이션입니다.

## Unity 버전
- 6000.1.0f1

## 폴더 구조
- `Assets/` : 에셋 및 소스 파일
- `Packages/` : 패키지 매니페스트
- `ProjectSettings/` : 프로젝트 설정

## 시작 방법
1. 이 저장소를 클론합니다.
2. Unity Hub에서 프로젝트 폴더를 열어주세요.
3. 필요에 따라 의존성 패키지를 자동으로 설치합니다.
